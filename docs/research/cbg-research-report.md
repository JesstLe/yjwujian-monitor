## 4. API接口

### 4.1 商品列表接口
```
GET https://yjwujian.cbg.163.com/cgi/api/get_aggregate_equip_type_list

参数:
- client_type: h5
- count: 15 (每页数量)
- page: 1 (页码)
- order_by: selling_time DESC
- query_onsale: 1
- kindid: 3 (3=英雄皮肤, 4=兵器皮肤, 5,6=道具)
- exter: direct
- page_session_id: [session id]
- traffic_trace: {"field_id":"","content_id":""}
```

### 4.2 登录信息接口
```
GET https://yjwujian.cbg.163.com/cgi/api/get_login_info
```

### 4.3 推荐接口
```
POST https://yjwujian.cbg.163.com/cgi-bin/recommend.py

参数:
- client_type: h5
- act: recommd_by_role
```

## 6. 盯盘系统功能建议

基于以上调研，建议盯盘系统实现以下核心功能：

### 6.1 P1 - 必须实现
1. **商品搜索与监控**: 支持按名称、分类、稀有度搜索商品
2. **价格预警**: 设置目标价格，低于目标时触发通知
3. **实时监控**: 定时轮询商品价格（建议5分钟间隔）

### 6.2 P2 - 重要功能
1. **价格历史图表**: 展示7/30/90天价格趋势
2. **收藏夹分组**: 按英雄或自定义分类管理监控商品
3. **市场概览**: 展示热门商品和价格变动趋势

### 6.3 P3 - 增强功能
1. **多渠道通知**: 浏览器推送、邮件、Webhook
2. **星格分析**: 基于星格属性的价格预测
3. **自动购买**: 满足条件时自动下单（需谨慎实现）

## 7. 技术注意事项

1. **API限流**: 网易可能对API进行限流，需要合理设置请求间隔
2. **登录状态**: 部分功能需要登录，需要处理Session过期
3. **数据一致性**: 价格可能有延迟，需要显示"最后更新时间"
4. **反爬策略**: 避免频繁请求，建议使用代理或分布式爬取

---

**调研结论**: 藏宝阁API结构清晰，数据字段完整，适合开发价格监控系统。建议采用Node.js + TypeScript技术栈，使用SQLite本地存储，实现轻量级桌面/Web应用。
